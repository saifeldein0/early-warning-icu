# ICU Risk Prediction Website

This public website package is a runnable demonstration of the Early Warning ICU dashboard.
It preserves the dashboard workflow and input schema, but replaces private deployment dependencies
with synthetic demo data and deterministic heuristic scores.

## What is included

- A Flask dashboard for hematology and non-hematology ICU risk review
- Synthetic export files under `demo_data/`
- A sample-data generator script
- The same input schema used by the internal application

## What is not included

- Real patient data
- Trained model binaries
- Hospital integration code
- Deployment credentials or internal infrastructure details

## Quick start

```bash
cd Website
pip install -r requirements.txt
python create_sample_data.py
python app.py
```

The app runs at `http://localhost:5000`.

## Demo behavior

- By default the public release runs in `EWS_PUBLIC_DEMO=1`
- The dashboard reads the latest synthetic export from `Website/demo_data/`
- Risk scores are generated by a deterministic heuristic over synthetic vitals, labs, and medication histories
- The manual refresh button reloads the latest synthetic export

## Input schema

The demo files use the same key columns as the private deployment:

- Identity and context: `PATIENT_NAME`, `MRN`, `LOCATION`, `ROOM`, `ADMISSION_DATE`, `ADMISSION_ORDER`
- Vital signs: `HEART_RATE`, `PULSE_OXIMETRY`, `TEMPERATURE`, `SYSTOLIC_BLOOD_PRESSURE`, `MEAN_ARTERIAL_PRESSURE`, `DIASTOLIC_BLOOD_PRESSURE`, `RESPIRATION_RATE`
- Labs: `AST_RESULT`, `CREATININE_RESULT`, `TOTAL_BILIRUBIN_RESULT`, `DIRECT_BILIRUBIN_RESULT`, `POTASSIUM_RESULT`, `HEMOGLOBIN_RESULT`, `LEUKOCYTE_COUNT_RESULT`, `ABSOLUTE_NEUTROPHILS`, `PLATELET_COUNT_RESULT`, `PROTHROMBIN_CONCENTRATION`
- Medications: `ANTIBIOTICS`, `NEUROLOGY_DRUGS`, `CARDIOLOGY_DRUGS`, `FUNGAL_DRUGS`

## Notes

- The public website is meant for code review and workflow demonstration, not clinical use.
- If private models are added later, the app can be adapted back to model-backed inference by disabling demo mode and restoring the model artifacts.
